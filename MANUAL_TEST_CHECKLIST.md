# Manual Test Checklist

Use this after automated checks when you want a visual gameplay pass in the editor or exported build.

- [ ] Start a new game.
- [ ] Fight at least one enemy.
- [ ] Open inventory and equip an item.
- [ ] Confirm every floor shows a golden artifact room and a brown shop room.
- [ ] Clear a floor.
- [ ] Open a chest.
- [ ] Take the normal exit.
- [ ] Take the elite exit.
- [ ] Clear the final floor and confirm the result screen appears.
- [ ] Die and confirm the active save slot is removed.
